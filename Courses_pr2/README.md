# Электронный журнал оценок студентов

Модульный фреймворк на Node.js с контейнером внедрения зависимостей и динамической загрузкой модулей по конфигурации.

## Структура

- `src/` — ядро (контейнер DI, загрузчик модулей, точка входа, логгер).
- `modules/` — расширяемые модули: `core`, `logging`, `validation`, `export`, `report`, `di-demo`, `lifetime-demo`, плюс `circular-a/b` для проверки циклов.
- `config/modules.json` — список включённых модулей.
- `test/` — испытания загрузчика.

## Запуск

```bash
npm install
npm start
npm test
```

После `npm start` рядом появится файл `grades.txt` с экспортированными оценками.

## Как добавить свой модуль

1. Создать файл в `modules/` с экспортом по контракту (`name`, `version`, `contractVersion`, `requires`, `register`, `init`).
2. Дописать имя файла в `config/modules.json`.

Код в `src/` менять не нужно.
